
**インストール（開発）**
1. Chrome の拡張機能ページを開く（`chrome://extensions/`）。  
2. 右上の「デベロッパーモード」を有効化。  
3. 「パッケージ化されていない拡張機能を読み込む」から本プロジェクトのルートフォルダを選択。  
4. ポップアップを開いて動作を確認。

**注意点と運用**
- `storage.sync` は容量制限があるため、設定やタグなど小さなデータのみを同期対象にしてください。大きなデータは自動的に `storage.local` に退避します。  
- インポートは既存データを上書きするため、事前にエクスポートしてバックアップを取ることを推奨します。  
- 開発用のテストユーティリティ（`simulateLargeSyncSave`）は本番では削除してください。  
- 権限は最小化すること。ホスト権限は `optional_permissions` としてユーザーに明示的に要求します。

**テスト**
- `tests/manual_test_plan.md` に手動テスト手順を記載しています。まずは同期失敗・チャンク化・インポート競合のシナリオを確認してください。
