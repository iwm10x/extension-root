# Web漫画 管理（Chrome拡張）

**概要**  
ページ情報を取得して「本棚」「お気に入り」を管理する軽量な Chrome 拡張です。タグ履歴、ホストごとの自動取得、`storage.sync` と `storage.local` のハイブリッド保存、エクスポート/インポート、競合解決ダイアログ、仮想リストによる大量データ対応を備えています。

**主な機能**
- ページからタイトル・サムネイル・URL・サイト名を取得（`scripting.executeScript`）
- 本棚とお気に入りの追加・編集・削除・移動
- タグ履歴と候補表示、デフォルトタグボタン
- ホスト権限管理（optional permissions を利用）
- `storage.sync` を小さなメタデータに利用、`storage.local` を本体データに利用するハイブリッド保存
- `saveHybridWithFallback` による同期の再試行とローカルフォールバック
- エクスポート/インポート（JSON）と競合解決ダイアログ
- 仮想リストによる大量アイテム表示の最適化
- 障害ログ（`mangaErrorLogs`）の蓄積とエクスポート対応
